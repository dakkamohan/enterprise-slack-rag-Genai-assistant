# enterprise-slack-rag-Genai-assistant

This repository contains an end-to-end serverless AI assistant built on AWS, designed to integrate seamlessly with Slack Slash Commands.
The solution leverages Retrieval-Augmented Generation (RAG) to provide accurate, context-aware responses by combining enterprise data with foundation models hosted on Amazon Bedrock.

The architecture follows AI DevOps best practices, emphasizing security, scalability, cost efficiency, and loose coupling using event-driven services.
